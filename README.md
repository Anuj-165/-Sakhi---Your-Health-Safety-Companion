🌸 Sakhi (सखी) - Your Health & Safety Companion
Sakhi (meaning "female friend" in Hindi) is a dedicated wellness and safety platform designed specifically for women. It combines AI-driven health assistance, menstrual cycle tracking, educational resources, and community storytelling into a single, "calm-gradient" inspired interface.

✨ Features
🤖 AI Health Assistant (Sakhi Bot)
Beautiful & Realistic: A feminine-coded, interactive bot with "breathing" and "blinking" animations.

Instant Queries: Ask questions about reproductive health, hygiene, or general wellness.

Voice Integration: Visual voice waves that react when Sakhi is "speaking" or processing.

📅 Period Tracker
Monitor menstrual cycles with a clean, intuitive calendar.

Log symptoms and receive wellness tips based on cycle phases.

📚 Education & Community
Learn: A library of articles covering everything from PCOS to mental health.

Story Alert: A safe space for women to share their experiences and alert others about safety concerns.

Find Chemist: Integrated geolocation to find the nearest pharmacies and medical stores.

🔒 Secure Authentication
JWT Protected: Secure login and registration.

Dynamic Profile: An "Auth-aware" UI that transforms the login page into a personalized user profile once signed in.

FastAPI Backend: Powered by a high-performance Python backend with /user/me endpoint for real-time profile fetching.

🛠️ Tech Stack
Frontend
React 18 (TypeScript)

Tailwind CSS (Custom "Calm" and "Empowerment" gradients)

Framer Motion (Smooth UI transitions and Bot animations)

Heroicons (Clean, accessible iconography)

Axios (API communication)

Backend
FastAPI (Python)

SQLAlchemy (Database ORM)

JWT (Authentication)

🚀 Getting Started
1. Clone the repository
Bash
git clone 
cd sakhi-wellness
2. Frontend Setup
Bash
cd frontend
npm install
npm run dev
3. Backend Setup
Bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
🎨 Design Language
Sakhi uses a custom design system to ensure users feel safe and empowered:

Primary Gradient: #6366F1 (Indigo) to #EC4899 (Pink).

Calm Gradient: Soft background fades to reduce visual stress.

Typography: Poppins (Bold for headings, Regular for body).
